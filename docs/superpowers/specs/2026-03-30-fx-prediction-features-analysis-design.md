# 外汇预测业界特征分析设计

**创建日期**：2026-03-30
**设计者**：iFlow CLI
**项目**：FX Predict - 外汇智能分析系统
**版本**：1.0

## 执行摘要

### 分析目标
全面分析业界外汇预测使用的特征类别，识别当前系统缺失的特征，评估数据源可行性，为 FX Predict 系统提供更全面的决策支持。

### 分析范围
- 货币对：EUR/USD, USD/JPY, AUD/USD, GBP/USD, USD/CAD, NZD/USD
- 时间维度：长周期（宏观）、中周期（市场）、短周期（技术）
- 数据源：仅考虑免费数据源
- 更新频率：每日更新

### 关键发现
1. 当前系统覆盖技术分析层（35个指标 + 18个特征），但缺失其他4层
2. 宏观经济层特征预测价值最高，且数据源丰富（FRED等）
3. 市场情绪层（COT报告）提供独特的投机仓位视角
4. 跨市场相关性（黄金、原油）对商品货币影响显著

### 优先实施建议
- 短期（1-3个月）：FRED宏观经济特征 + Yahoo Finance跨市场数据
- 中期（3-6个月）：CFTC COT报告 + 高级技术形态识别
- 长期（6-12个月）：社交媒体情绪分析 + 深度学习模型

## 分析架构设计

### 分层特征框架

采用5层特征分类框架，从宏观到微观，从基本面到市场情绪：

```
┌─────────────────────────────────────────┐
│ 第5层：市场情绪层                       │
│ - 投机仓位、散户情绪、新闻情绪          │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第4层：技术分析层                       │
│ - 35个技术指标 + 18个工程特征           │
│ 【当前系统已实现】                      │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第3层：跨市场层                         │
│ - 商品价格、股指、货币对相关性          │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第2层：市场结构层                       │
│ - 流动性、买卖价差、市场参与者          │
└─────────────────────────────────────────┘
                  ↓
┌─────────────────────────────────────────┐
│ 第1层：宏观经济层                       │
│ - 利率、通胀、GDP、贸易、政策           │
└─────────────────────────────────────────┘
```

### 评估维度

每个特征使用以下5个维度评估：

| 维度 | 说明 | 评分标准 |
|------|------|---------|
| 预测价值 | 特征对汇率预测的理论贡献 | 1-5分，5分最高 |
| 数据可用性 | 免费数据源获取难度 | 高/中/低 |
| 更新频率 | 数据更新周期 | 实时/日内/日/周/月 |
| 实施难度 | 集成到系统的技术难度 | 低/中/高 |
| 相关性匹配 | 与当前6个货币对的相关性 | 1-5分，5分最高 |

### 对比评估矩阵

形成"现状-业界差距-优先级"矩阵，帮助决策：

| 特征类别 | 业界特征数 | 当前系统特征数 | 覆盖率 | 缺失率 | 优先级 |
|---------|-----------|---------------|--------|--------|--------|
| 宏观经济层 | 15 | 0 | 0% | 100% | 高 |
| 市场结构层 | 8 | 0 | 0% | 100% | 中 |
| 跨市场层 | 10 | 0 | 0% | 100% | 高 |
| 技术分析层 | 53 | 53 | 100% | 0% | - |
| 市场情绪层 | 12 | 0 | 0% | 100% | 中 |

## 分析范围和方法

### 分析范围

**货币对覆盖**：
- 当前支持的6个货币对：EUR/USD, USD/JPY, AUD/USD, GBP/USD, USD/CAD, NZD/USD
- 关键相关货币：EUR, USD, JPY, GBP（基准货币和报价货币）

**时间维度**：
- 长期：月度、季度经济指标（利率、通胀、GDP）
- 中期：周度、日度市场数据（跨市场相关性、市场结构）
- 短期：日度技术指标（当前系统已实现）

**业界方法覆盖**：
1. 传统宏观经济模型（货币主义、购买力平价、利率平价）
2. 统计套利模型（协整关系、均值回归）
3. 机器学习模型（随机森林、XGBoost、CatBoost）
4. 深度学习模型（LSTM、Transformer）
5. 混合模型（基本面 + 技术面 + 市场情绪）

### 分析方法

**1. 文献研究**
- 查阅学术论文：Journal of Finance, IMF Working Papers
- 分析量化交易机构白皮书
- 研究央行研究报告

**2. 特征工程分析**
- 识别每个特征类别的具体变量
- 分析特征计算方法和参数
- 评估特征与汇率的理论关联

**3. 数据源验证**
- 确认免费数据源（FRED, Yahoo Finance, CFTC等）
- 评估数据质量、历史长度、更新频率
- 测试数据获取的可行性

**4. 案例研究**
- 分析成功系统（Bloomberg, Reuters, TradingView）
- 研究开源项目（Quantopian, Backtrader）
- 对比不同策略的表现

### 数据源评估标准

**数据质量**（权重30%）：
- 数据准确性（错误率<1%）
- 数据完整性（缺失值<5%）
- 数据一致性（口径统一）

**历史长度**（权重20%）：
- 宏观数据：至少10年历史
- 市场数据：至少5年历史
- 情绪数据：至少2年历史

**更新频率**（权重20%）：
- 高频（日内）：实时价格、VIX
- 日频：经济指标、市场情绪
- 周频：COT报告、PMI
- 月频：CPI、GDP

**访问便利性**（权重20%）：
- API接口稳定性
- 数据格式友好度（JSON/CSV）
- 速率限制（调用次数/秒）
- 文档完整性

**成本**（权重10%）：
- 完全免费（10分）
- 免费但有速率限制（8分）
